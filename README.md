## 🛠️ Installation Steps

<p><b>1. Clone the repository</b></p>

```bash
git clone https://github.com/oktaviandwip/tickitz-frontend
```

<p><b>2. Install dependencies</b></p>

```bash
npm install
# or
yarn install
```

<p><b>3. Run the app</b></p>

```bash
npm run dev
# or
yarn dev
```
