## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/oktaviandwip/tickitz-frontend
```

2. Install dependencies

```bash
npm install
# or
yarn install
```

3. Run the app

```bash
npm run dev
# or
yarn dev
```
